# Video Motivasi: Hutan & Sungai

Generator video kata-kata motivasi berlatar suasana hutan dan sungai yang damai. Seluruh pemrosesan video dilakukan di browser dengan Canvas + MediaRecorder. Cocok untuk deployment di Vercel.

## Menjalankan

```bash
npm install
npm run build
npm start
```

Buka http://localhost:3000 dan klik "Buat Video".

## Deploy Vercel

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-80eaf338
```
